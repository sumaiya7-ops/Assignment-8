# 🚀 SkillSphere - Modern Learning Platform

SkillSphere is a modern online learning platform built with **Next.js, Tailwind CSS, and Better Auth**.  
It allows users to explore courses, view details, manage profiles, and authenticate using email/password or Google login.

## ✨ Features

- 🔐 User Authentication (Email/Password + Google OAuth)
- 📚 Dynamic Course Listing System
- 🔥 Trending Courses (Based on Ratings)
- 🔍 Search & Filter Courses
- 👤 User Profile Page
- ✏️ Profile Update Feature
- 📱 Fully Responsive UI (Mobile + Desktop)
- 🎨 Modern UI with Tailwind CSS
- ⚡ Fast Performance with Next.js App Router

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS + DaisyUI
- Better Auth
- MongoDB
- Lucide Icons

---

## 📂 Project Structure

app/ → Pages (Home, Courses, Profile)  
components/ → Reusable UI Components  
data/ → Static Course Data (courses.json)  
lib/ → Authentication & Helpers  
public/ → Images & Assets

## 🔐 Environment Variables

Create a `.env.local` file in the root directory:

```bash
MONGODB_URI=your_mongodb_connection
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
AUTH_SECRET=your_auth_secret
